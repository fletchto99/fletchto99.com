Pebble Smartwatch Development

I have written multiple apps for the Pebble Smartwatch platform. These apps are open source and are written in either javascript or C. Some of them rely on a server side to make API calls, which is written in PHP.

The first app is Pebble meetup. This app allows a user to view upcoming pebble meetups in their area. Through the use of C, JS, and Meetup.com's API I am able to fetch event and group information which is then displayed on the watch. Find out more here
