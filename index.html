Web Developer - OCDSB

2015-Present

Developer for the web application, MaMa, which manages evidence records for the teachers of the Ottawa Carleton District Schoolboard. In the first year of operations the system has had over 2 million evidence records entered!

With this application teachers are able to enter evidence and comment for assessments preformed within the classroom. It essentially replaces the need to record marks in a spreadsheet or on a piece of paper and allows teacher to conveniently have everything in one place. Unfortunately due to the nature of this application a preview is unavailable.

Pebble Smartwatch Development

2014-Present

I have written multiple apps for the Pebble Smartwatch platform. These apps are open source and are written in either javascript or C. Some of them rely on a server side to make API calls, which is written in PHP.

The first app is Pebble meetup. This app allows a user to view upcoming pebble meetups in their area. Through the use of C, JS, and Meetup.com's API I am able to fetch event and group information which is then displayed on the watch. Find out more about Meetup here

The second app is uCard Manager, which allows a university of Ottawa student to check their meal plan balance directly from their watch. Find out more about uCard Manager here

My most recent app called "devboard" allows developers to easily check the status of their pebble projects, directly from their watch! Find out more aobut devboard here
