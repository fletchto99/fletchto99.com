Web Developer: OCDSB

2015-Present

Developer for the web application, MaMa, which manages evidence records for the teachers of the Ottawa Carleton District Schoolboard. In the first year of operations the system has had over 2 million evidence records entered!

With this application teachers are able to enter evidence and comment for assessments preformed within the classroom. It essentially replaces the need to record marks in a spreadsheet or on a piece of paper and allows teacher to conveniently have everything in one place. Unfortunately due to the nature of this application a preview is unavailable.

Pebble Smartwatch Development

2014-Present

I am an active community developer for the Pebble Smartwatch platform. As an enthusiast for the platform I have created multiple app, added to documentation, created developer libraries andhave contributed quite a bit to some of their open source libraries. Here's a few examples of my works:

  • My Pebble Apps
    • Meetup: Is my most popular Pebble app enabling users to look for local meetup events from the website Meetup.com
    • Devboard: Enables Pebble developers to check the status of their Pebble projects directly from their watch. Including # of installs, date published, features enabled, etc.
    • uCardManager: Allows students at the University of Ottawa to view their remaining studentcard and mealplan balances.
  • My Pebble Contributions
    • PHPebbleTimeline: Is an open source library I created to enable developers to push Timeline pins to the Pebble Smartwatch using PHP. It's source is available here.
    • PebbleJS: Contributor for the open source PebbleJS library, allowing users to create an app for the Pebble platform using Javascript. As a contributor I have work quite a bit on the menu items.
    • Slate: Contributed to the open source Slate library which allows developers to easily create configuration pages for their Pebble apps. One of my major contributions is the responsive tabbed navigation.

#TimelineChallenge Winner: Meetup

2015

My Pebble app Meetup won week 6 of the TimelineChallenge for being one of the best apps to integrate with the new timeline feature on the Pebble Time. The app allows users to look for events in the area and then pin them to the watch's timeline so that the user doesn't forget about the event. Since it's release Meetup has had over 850 downloads! Be sure to check out this awesome #timelinechallenge blog post Pebble did on my app!
